Mindmap Course Format - displays course content as interactive,
personalized midmap.

It uses Freeplane Java Applet to display map in browser.

Compatibility with Moodle 3.0

1.1.5.1
Yet another improvement of nodes folding

1.1.5
Fixed syntax of map file
Improved display of course page in editing mode
Fixed bug where some modules could be displayed as folded, while they
should be unfolded
Center node of map is now course shortname instead of fullname

1.1.4
Moodle 2.9 compatibility

1.1.3
Fixed display of Chinese characters

1.1.2
Fixed display of national characters

1.1.1
